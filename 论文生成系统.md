<pre class="hljs"><code><div><span class="hljs-section"># 实训项目：在线论文文档自动生成工具开发</span>
<span class="hljs-section">## 一、项目整体需求概述</span>
本项目需开发一套Web端在线论文编辑与标准化文档生成系统，主要用于辅助学生规范化完成课程论文、毕业论文、项目设计类论文的在线撰写、版式排版与文档导出。系统设置管理员、学生、教师三类用户角色，分别赋予模板管理、论文编辑导出、课程论文线上批阅等对应权限，实现论文模板统一配置、网页端图文表格参考文献编辑、全文在线预览、docx与PDF格式文件导出、课程论文批阅留痕等完整业务流程。

<span class="hljs-section">## 二、系统角色与权限划分</span>
<span class="hljs-section">### 1. 系统管理员</span>
负责学院信息维护、多类型论文模板新增、编辑、上下架管理，管控模板分类与版本，具备后台基础数据管理权限，不可直接编辑学生论文内容。

<span class="hljs-section">### 2. 学生用户</span>
可按需筛选学院、论文类型选用对应模板，在线完成论文内容编写、素材上传、草稿保存、全文预览，导出Word与PDF文档，课程论文可提交至教师端等待批阅。

<span class="hljs-section">### 3. 教师用户</span>
仅针对课程论文拥有批阅权限，可查看学生提交的论文内容，添加批注修改意见、填写评语与评定成绩，支持退回论文供学生修改重提，留存历次批阅历史记录。

<span class="hljs-section">## 三、管理员模板管理模块功能</span>
<span class="hljs-bullet">1.</span> 支持按学院维度对论文模板进行分类管理，可新增、删除、编辑院系信息，绑定对应院系专属论文格式模板。
<span class="hljs-bullet">2.</span> 可自定义发布多类型模板，包含毕业论文模板、课程论文模板、项目论文模板，支持模板启用与停用下架操作。
<span class="hljs-bullet">3.</span> 模板内置固定规范结构，预设论文封面、原创声明、中英文摘要、关键词、多级章节正文、附录、参考文献等固定版式，统一约束字体、行距、段落缩进、页边距、页眉页脚、页码等排版格式。
<span class="hljs-bullet">4.</span> 支持模板在线预览与内容修改，模板更新后，学生新建论文将自动套用最新模板格式。

<span class="hljs-section">## 四、学生在线论文编辑核心功能</span>
<span class="hljs-bullet">1.</span> 结构化章节编辑：依托选定模板大纲分章节填写正文内容，编辑器支持标题层级设置、文字加粗、对齐方式、段落缩进等基础排版操作。
<span class="hljs-bullet">2.</span> 素材插入功能：支持本地上传图片并嵌入文档指定位置，可在线插入、编辑自定义表格，适配论文数据表格、学术三线表等使用场景。
<span class="hljs-bullet">3.</span> 参考文献管理：独立模块逐条录入文献作者、标题、发表刊物、出版年份、页码等信息，系统依据模板格式自动统一排版参考文献列表，支持正文正文引用标注与文献条目关联。
<span class="hljs-bullet">4.</span> 云端草稿存储：编辑过程支持自动保存与手动保存草稿，学生可多次进入项目查看、续写、修改历史论文文档。

<span class="hljs-section">## 五、全文预览与多格式文档导出</span>
<span class="hljs-bullet">1.</span> 网页端全文预览：编辑完成后一键生成整篇论文网页预览页面，完整展示封面、各级章节、图片、表格、参考文献整体排版效果，支持分页浏览整篇文档。
<span class="hljs-bullet">2.</span> 文档导出功能：按照所选模板既定格式，一键导出标准docx格式Word文档；同时支持生成并导出PDF格式文件，导出文件排版规范、图文资源完整无错乱。

<span class="hljs-section">## 六、课程论文教师批阅专属功能</span>
<span class="hljs-bullet">1.</span> 学生完成课程论文编辑后可提交给对应授课教师，提交后文档锁定，仅教师可执行批阅或退回操作。
<span class="hljs-bullet">2.</span> 教师在线查看论文预览内容，可在文档对应位置添加批注、修改建议，填写整体评语并评定论文分数与等级。
<span class="hljs-bullet">3.</span> 系统留存每一次提交、批阅、退回版本记录，学生可查看全部批注内容，根据修改意见调整论文后重新提交复审。

<span class="hljs-section">## 七、实训开发要求</span>
采用Web前后端分离架构完成系统整体开发，实现用户登录鉴权与权限控制、模板配置管理、富文本在线编辑、文档结构解析与文件生成、文件下载、课程论文批阅流程流转等核心业务逻辑，最终交付可部署运行的完整Web应用，搭建从模板分发、在线写作、预览核查、文档导出到课程论文线上批改的一体化论文生成工具。
