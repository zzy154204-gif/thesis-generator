\subsection{性能结论}

\begin{itemize}
    \item 单机 50 并发以下表现良好，核心接口 QPS >300
    \item 95\% 请求在 200ms 内完成
    \item Docker 三容器内存峰值约 720MB，适合 2GB+ 服务器
    \item LibreOffice PDF 导出耗时 1-5 秒（取决于内容复杂度）
\end{itemize}
