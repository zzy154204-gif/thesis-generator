\section{前端界面交互架构}
本系统前端交互自顶向下划分为四层：用户操作层、业务交互层、状态管理层、数据服务层
\begin{itemize}
	\item 用户操作层: 捕获用户点击、输入、拖拽、快捷键等原始操作事件
	\item 业务交互层: 根据用户角色和当前状态，执行对应的业务逻辑
	\item 状态管理层: 维护全局和局部状态，驱动视图更新
	\item 数据服务层: 对封装 API 的调用，处理请求/响应/异常
\end{itemize}
\section{页面级别前端业务交互详述}
\subsection{管理员端界面}
\begin{itemize}
	\item 加载列表 触发方式：页面进入、点击搜索 交互反馈：显示loading骨架屏 数据：GET并渲染表格
	\item 新增学院 触发方式：点击学院管理页“新增”按钮 交互反馈：弹出模态框并聚焦第一个输入框 数据：填写表单并POST
	\item 编辑学院 触发方式：点击学院管理行内“编辑”按钮 交互反馈：弹出模态框 数据：修改并PUT
	\item 删除学院 触发方式：点击学院管理行内“删除”按钮 交互反馈：弹出警告确认框 数据：确认并DELETE
\end{itemize}
\subsubsection{管理员端核心：模板管理界面}
\begin{itemize}
	\item 模板类型tab切换 触发方式：点击对应类按钮 交互反馈：样式切换激活并刷新卡片 数据：GET
	\item 模板启用与停用 触发方式：点击模板卡片底部开关 交互反馈：状态标签实时变化，开关动画 数据：PATCH
	\item 模板预览 触发方式：点击模板卡片“预览”按钮 交互反馈：新窗口展示HTML预览 数据：GET
	\item 模板编辑与新建 触发方式：点击按钮 交互反馈：跳转至编辑器对应功能路径
\end{itemize}
\subsection{教师端界面}
\begin{itemize}
	\item 加载列表 触发方式：页面进入 交互反馈：显示loading 数据：Get渲染表格
	\item 筛选 触发方式：选择课程、学生姓名、时间范围 交互反馈：列表即时刷新 数据：带查询参数的GET
	\item 进入批阅 触发方式：点击论文列表行内“批阅”按钮 交互反馈：跳转至批阅页
\end{itemize}
\subsubsection{教师端核心：批阅界面}
\begin{itemize}
	\item 选文批注 触发方式：预览区拖拽选中文本，在浮动工具栏出现“批注”图标 交互反馈：输入框聚焦并提示，点击“提交”按钮上交批注
	\item 查看批注定位 触发方式：点击侧边栏批注条目 交互反馈：预览区自动滚动至对应位置
	\item 批注修改 触发方式：鼠标悬停批注条目后点击“编辑” 交互反馈：批注变为可编辑状态，修改后保存并更新侧边栏
	\item 批注删除 触发方式：鼠标悬停批注条目后点击“删除” 交互反馈：二次确认后删除，侧边条目消失
	\item 总评填写 触发方式：在评语文本框中输入 交互反馈：实时统计字数，超过 500 字限制输入
	\item 状态暂存 触发方式：点击对应“暂存”按钮 交互反馈：保留目前批注、评语、打分状态，论文批改状态不变
	\item 论文判定 触发方式：点击“退回”或“通过”按钮 交互反馈：校验填写项后二次确认，改变论文批改状态并返回列表
\end{itemize}
\subsection{学生端论文览阅界面}
\begin{itemize}
	\item 打开预览 触发方式：点击顶栏“预览”按钮	交互反馈：新标签页展开，显示完整论文的 A4 风格排版
	\item 分页浏览 触发方式：滚动鼠标或点击翻页按钮 交互反馈：页码更新，页面切换
	\item 导出交互 触发方式：点击导出 交互反馈：提供pdf与docx选项，导出完成自动下载文件
\end{itemize}
\subsubsection{学生端核心：论文编辑界面}
\begin{itemize}
	\item 新建章节 触发方式：点击“添加章节”按钮 交互反馈：弹出对话框输入章节标题并选择层级（1级/2级/3级），确认后在当前选中节点下创建及刷新树
	\item 切换章节 触发方式：点击章节树节点	交互反馈：自动保存当前章节，编辑器加载新章节内容并滚动到顶部
	\item 重命名章节 触发方式：章节节点右键菜单“重命名” 交互反馈：弹出输入框预填原标题，修改后确认更新节点名称，调用接口保存
	\item 删除章节 触发方式：章节节点右键菜单“删除” 交互反馈：二次确认弹窗，删除章节及其子章节，刷新树并跳转回父章节
	\item 章节排序 触发方式：拖拽节点上下移动 交互反馈：节点位置变化，自动保存树的更新
	\item 章节变级 触发方式：拖拽节点至另一个节点内部（缩进） 交互反馈：节点父级关系变化，自动保存更新
	\item 文字排版 触发方式：选中文字并点击工具栏按钮（B/I/U/H1/H2等）	交互反馈：实时应用格式，编辑器内容更新
	\item 图片插入 触发方式：点击工具栏“图片”按钮并选择本地图片	交互反馈：编辑器光标处插入图片（自适应宽度）
	\item 粘贴图片 触发方式：从剪贴板粘贴图片（Ctrl+V） 交互反馈：自动触发图片上传流程，与点击上传体验一致
	\item 表格插入 触发方式：点击“表格”按钮，选择行列数（3×4 网格选择器）	交互反馈：在光标处插入指定行列的空表格，支持合并/拆分单元格
	\item 引用标注 触发方式：光标定位后点击“插入引用”按钮，选择参考文献	交互反馈：在光标处插入 [n] 标记，文献库中该条目标记“已被引用”
\end{itemize}

\section{交互反馈与异常处理}
\subsection{用户操作反馈规范}
\begin{itemize}
	\item 操作成功：绿色 Toast（顶部居中，自动消失） 如：保存、新增、编辑、删除成功
	\item 操作失败：红色 Toast（顶部居中，自动消失） 如：请求失败、校验不通过
	\item 确认操作：模态对话框（带“确定/取消”按钮）  如：删除、退回、通过、提交
\end{itemize}
\subsection{异常场景应对策略}
\begin{itemize}
	\item 网络断开:toast显示“网络连接异常，请检查网络后重试”
	\item 401 Unauthorized:显示“请求超时，请稍后重试”
	\item 403 Forbidden:显示“您没有权限执行此操作”，并禁用按钮
	\item 500 Internal Error:显示“服务器错误，请稍后重试”
	\item 表单校验失败:输入框红色边框 + 下方错误文字实时提示
\end{itemize}
\subsection{人性化防触设计}
\begin{itemize}
	\item 防误删：所有删除操作均需二次确认，确认对话框明确显示被删除对象的标识信息
	\item 快速连点：按钮点击后添加 500ms 防抖，避免短时间内重复触发
	\item 重复提交：提交按钮点击后立即置灰 + 显示“提交中…”，响应完成后恢复
	\item 未保存退出：编辑器页面检测到脏数据时，事件弹窗提醒
\end{itemize}
\subsection{UI规范杂项}
\begin{itemize}
	\item 按钮位置	确认类按钮（确定/保存/提交）在右侧，取消类按钮在左侧
	\item 颜色语义	绿色=成功/通过，红色=危险/删除/退回，蓝色=主要操作/链接，橙色=警告
	\item 字数限制	表单输入框超限时实时提示，评语文本框显示剩余字数（0/500）
	\item 日期时间	统一格式 YYYY-MM-DD HH:mm，显示在操作记录和列表时间列
\end{itemize}
