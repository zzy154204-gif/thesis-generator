对于各学院的教师以及系统的管理员，本系统需要为其设计教师需要的批阅功能，以及管理员对全系统教师、学生、论文及其模板的管理
\section{管理员端}
\subsection{学院信息管理}
为区分系统下不同学院论文模板规范，管理员端需要管理下属学院信息的功能
\begin{itemize}
	\item 列表展示：以表格分页展示学院ID、名称、代码、创建时间
	\item 新增学院：弹窗录入学院名称（必填，不可重名）和代码
	\item 编辑学院：支持修改学院名称及代码
	\item 删除学院：仅当该学院下无关联模板与用户时允许删除，删除前需弹出二次确认框
\end{itemize}
\subsection{论文模板统一配置}
管理员统一配置论文格式、结构及封面字段，学生新建论文时以此为标准
\begin{itemize}
	\item 分类展示：使用Tab切换展示“毕业论文”、“课程论文”、“项目论文”三类模板卡片
	\item 卡片信息：底部开关控制模板上/下架，明显位置提示版本号
	\item 预览：支持预览模板整体排版样式（封面+正文范例）
	\item 模板编辑器：进入编辑页后，进行对模板的封面配置、样式配置以及结构配置
\end{itemize}

\section{教师端}
\subsection{课程论文批阅流程}
教师对学生论文评阅，进行论文状态流转
\begin{itemize}
	\item 待批列表：教师首页/批阅菜单展示“待批阅”论文列表（含学生姓名、学号、提交时间）
	\item 锁定机制：学生提交后，论文自动锁定为只读，学生端禁止编辑
	\item 在线查看：教师可打开网页端预览（同学生预览效果），查看完整论文图文
	\item 评语与成绩：教师在独立区域填写整体评语并评定分数（百分制）与等级（优/良/中/及格/不及格，可按分数区间自动映射）
	\item 退回操作：若论文不合格，教师点击“退回”，论文状态变为待修改，解锁编辑权限，学生可修改后重提
\end{itemize}
\subsection{论文检查交互系统}
教师在批阅过程中，实现学生可见的交互讨论
\begin{itemize}
	\item 批注显示: 批注在页面右侧以侧边栏形式聚合显示，点击批注可定位到正文对应位置
	\item 异常阻断: 若教师未填写评语直接提交“通过”，系统需弹窗提示“请填写评语后再提交”
\end{itemize}
