对于各学院的教师以及系统的管理员，本系统需要为其设计教师需要的批阅功能，以及管理员对全系统教师、学生、论文及其模板的管理
\section{管理员端}
\subsection{学院信息管理}

为区分系统下不同学院论文模板规范，管理员端需要管理下属学院信息的功能

\begin{itemize}
	\item 列表展示：以表格分页展示学院ID、名称、代码、创建时间;支持分页（每页10/20/50条可选）
	\item 新增学院：弹窗录入学院名称（必填，不可重名）和代码
	\item 搜索筛选: 顶部提供搜索框，支持按学院名称或代码进行模糊搜索
	\item 编辑学院：点击表格行末“编辑”按钮弹出模态框，支持修改学院名称及代码
	\item 删除学院：仅当该学院下无关联模板与用户时允许删除，删除前需弹出二次确认框；若该学院已被模板引用，系统提示“该学院已被X个模板引用，请先解除关联或迁移模板后再删除”并阻止删除操作
\end{itemize}

\subsection{论文模板统一配置}

管理员统一配置论文格式、结构及封面字段，学生新建论文时以此为标准

\begin{itemize}
	\item 分类展示：使用Tab切换展示“毕业论文”、“课程论文”、“项目论文”三类模板卡片
	\item 卡片化信息：每个模板以卡片形式呈现，卡片内容包含：模板名称、所属学院、模板类型标签、版本号、状态标签（启用/停用）、更新时间
	\item 模板预览：支持预览模板整体排版样式（封面+正文范例）
	\item 新增模板: 卡片列表顶部提供“新建模板”按钮，点击后跳转至模板编辑器（空白配置）
	\item 模板编辑器：进入编辑页后，进行对模板的封面配置、样式配置以及结构配置
\end{itemize}
\subsection{模板编辑详细要素}
\begin{itemize}
	\item 模板名称: 如“XX大学本科毕业论文模板”
	\item 所属学院: 下拉选择（数据来源于学院管理)
	\item 模板类型: 毕业论文/课程论文/项目论文（单选)
	\item 版本号: 自动递增，格式 (主版本.次版本.修订号)
	\item 模板描述: 说明模板适用场景
	\item 封面字段配置: 采用左右双栏拖拽设计;左侧字段池固定字段库(论文标题、学生姓名、学号、专业、学院、指导教师、提交日期、摘要、关键词、原创声明),右侧已选字段设计为当前模板封面实际使用的字段列表
	\item 样式配置: 从可选项选择正文字体、正文字号&行距、首行缩进、页边距（上/下/左/右）、页眉内容&字号、页码位置&起始页
	\item 章节配置： 原创声明&中英文摘要开关，标题层级选择，附录与参考文献设置
\end{itemize}
\subsection{模板更新规则}
\begin{itemize}
	\item 模板套用： 新建论文时，系统自动关联该类型下状态为“已启用”且最新的模板版本
	\item 已有论文: 模板更新后，已创建的论文保持原有格式不变，仅新创建的论文使用新模板
	\item 版本记录:	每次修改保存生成一条版本记录（template_version 表），可回滚至任意历史版本
\end{itemize}
\section{教师端}

\subsection{课程论文批阅流程}

教师对学生论文评阅，进行论文状态流转

\begin{itemize}
	\item 待批列表：教师首页/批阅菜单展示“待批阅”论文列表（含学生姓名、学号、提交时间）
	\item 锁定机制：学生提交后，论文自动锁定为只读，学生端禁止编辑
	\item 在线查看：教师可打开网页端预览（同学生预览效果），查看完整论文图文
	\item 暂存功能: 教师可随时点击“暂存”保存评语和评分，但论文状态不变
	\item 评语与成绩：教师在独立区域填写整体评语并评定分数（百分制）与等级（优/良/中/及格/不及格，可按分数区间自动映射）
	\item 退回操作：若论文不合格，教师点击“退回”，论文状态变为待修改，解锁编辑权限，学生可修改后重提
\end{itemize}

\subsection{论文检查交互系统}

教师在批阅过程中，实现学生可见的交互讨论

\begin{itemize}
	\item 添加批注:教师在预览区选中一段文字/段落后点击“添加批注”按钮，输入批注内容
	\item 批注显示: 批注在页面右侧以侧边栏形式聚合显示，点击批注可定位到正文对应位置
	\item 异常阻断: 若教师未填写评语直接提交“通过”，系统需弹窗提示“请填写评语后再提交”
\end{itemize}
